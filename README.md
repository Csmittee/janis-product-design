# Janis Product Design

Hardware product design repo for:
- Vending Machine (VM-01, VM-02, VM-03)
- Pilates AI Reformer (PR-01, PR-02, PR-03, PR-04)

## Stack
Claude + AdamCAD + OpenSCAD + Vectary + Canva

## Structure
- /COMMON — shared components across all products
- /vending-machine — VM variants
- /pilates-reformer — PR variants
- /exports — supplier ready files
- /prompts — all AdamCAD prompts
- /drawings — 2D technical drawings
- /renders — client presentation images

# VM-01 Base Vending Machine — Prompt v1
Date: 2026-06-27

## Brief
Compact smart vending machine, stainless steel skin,
modern sleek rounded corners like monitor/microwave.
Proto uses L-brackets for structure.

## Dimensions
- Total width: 620mm
- Total height: 800mm  
- Total depth: 410mm
- Corner radius: 20mm
- Leg height: 150mm, 25mm OD round x4

## Spring/Tray System
- Spring OD: 66mm, Length: 390mm, Pitch: 25mm
- 2 trays, 5 lanes each, gap 20mm between springs
- Tray height: 116mm (66mm OD + 50mm clearance)
- Lane partitions: 40mm high
- Tray slides in/out, lift-to-remove stopper

## Compartments
- Left: product compartment 416mm wide
- Right: system compartment 185mm (7" TFT at 30deg + electronics)
- Divider: 20mm

## Doors (4 total)
1. Main door: full face left compartment, hinge far left
2. Exit door: 400mm wide, push-in, hinge top, at 400mm from floor
3. Top acrylic: lift-open display area above right compartment
4. Rear service door: full size back of dashboard zone, one side hinge

## Drop Zone
- 175mm depth in front of spring tray
- Above exit door

## Screen
- 7" TFT, 30deg angle from horizontal
- Top edge at 40% height from bottom = 320mm from floor
- Right compartment

## Structure
- Stainless steel skin
- L-brackets for prototype structure
- Future: welded reinforcement shell
- Main board access: side or back of right compartment
